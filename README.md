# 📊 Análise de Perfil de Clientes - Campanha de Marketing

## 📌 Contexto
Este projeto tem como objetivo analisar o perfil de clientes em uma campanha de marketing de um produto de investimento.

A campanha teve duração de 3 meses e abrangeu clientes em todo o Brasil, com o intuito de identificar padrões de comportamento e melhorar a eficiência de futuras campanhas.

---

## 🎯 Problema de negócio
Identificar o perfil dos clientes com maior propensão à conversão, permitindo campanhas mais assertivas, redução de custos e aumento de resultados.

---

## 📁 Base de dados
Arquivo: `analise-clientes-marketing.xlsx`

## 📌 Observação
Este projeto utiliza dados fictícios para simular um cenário real de análise de dados.

---

## 📌 Fonte dos dados
Base de dados utilizada para fins educacionais, proveniente do curso da Preditiva Analytics.  
🔗 https://www.preditiva.ai/

---

## 🔍 Etapas da análise
- Limpeza e organização dos dados  
- Análise exploratória (EDA)  
- Identificação de padrões  
- Geração de insights  

---

## 📊 Principais insights
- A taxa de conversão foi de aproximadamente 49%  
- A idade não apresentou impacto significativo na conversão  
- Estudantes e aposentados apresentaram maior taxa de conversão  
- Perfis fora do mercado tradicional demonstraram maior propensão à compra  

---

## 📈 Impacto para o negócio
Os insights gerados permitem:
- Melhor segmentação de clientes  
- Otimização de campanhas de marketing  
- Aumento da taxa de conversão  
- Tomada de decisão baseada em dados
